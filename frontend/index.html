Створення гаманця

Ми згенеруємо пару ключів ECDSA (P-256). Адреса = хеш public key.

Якщо пароль порожній — ключ збережеться без шифрування (тільки для MVP).

Імпорт

Встав Private Key (PEM) або імпортуй із локального vault (якщо ти вже створював гаманець у цьому браузері).

Для простоти MVP: можна імпортувати PEM приватного ключа.
